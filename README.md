## Overview

**py.ref** is a self-contained, searchable Python reference covering everything from beginner syntax to professional-level topics — concurrency, decorators, testing, databases, web development, data science, and Git. It's built as a single portable HTML file with a terminal-inspired dark theme: no build step, no npm install, no external CDN calls. Open it and it works, online or off.

Built as part of an ongoing self-directed Computer Science curriculum, modeled after the structure of top university coursework.

<br>

## Usage

No installation required — clone and open.

```bash
git clone https://github.com/KhielDecena/py.ref.git
cd py.ref
open index.html      # macOS
# or just double-click the file / drag it into any browser
```

Want it hosted? Drop it straight into **GitHub Pages**, **Vercel**, or **Netlify** — it's a static file, so deployment is instant.

<br>

## Tech Stack

Vanilla **HTML5**, **CSS3**, and **JavaScript** — no frameworks, no bundlers, no package manager. This is a deliberate constraint: the site should remain a single, portable, dependency-free artifact that works anywhere, forever.
